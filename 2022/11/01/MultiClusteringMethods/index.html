kMeans原型聚类算法分析

$\qquad$kMeans聚类算法是一种非监督学习的分类算法，利用相似性度量方法来衡量数据集中所有数据之间的关系，将关系比较密切的数据划分到一个集合中。

算法原理分析

$\qquad$通过循环，不断迭代类中心点$\muk$，计算各个对象到新的类中心点的距离$d{(i)}$，并根据距离最近的原则重新归类，当类内距离最小、类间距离最大时，即可停止迭代（使用中，常常会限定迭代次数，防止陷入死循环。当达到预先设定得循环次数或类中心点不再发生变化时，最后一次迭代得到的结果，即为最终聚类结果）。

$\qquad$每个样本点$x{(i)}$与类内中心$\mu{(j)}$，其中$i$代表第$i$个样本点，$j$代表第$j$类，一般采用欧式距离计算样本点与各类中心点的距离，公式如$(1)$

$$\tag{1}
d^{(i)} := ||x^{(i)} - \mu^{k}||_2
$$

$\qquad$通过计算可以得到样本点$x{(i)}$到达各类中心点$\muk$的距离，然后取距离数组中$d{(i)}$的最小值，并把样本点$x{(i)}$划分为第$k$类，使用向量$c^{(i)}$表示每个样本点被划分到哪一类，公式如$(2)$

$$\tag{2}
c^{(i)} := j \quad minimizes \quad d^{(i)}
$$

$\qquad$将每个样本点分好类后，对每一类的样本点重新求其类内中心点$\mu_k$并更新，$C_k$表示一类内的样本点数，公式如$(3)$

$$\tag{3}
\mu_k:=\frac{1}{|C_k|} \sum_{i}^{C_k} x^{(i)}
$$

$\qquad$kMeans聚类算法比较依赖初始化的类中心点以及分类簇数K，再设定开始的类中心点时应该随机选择样本点作为类中心点，簇数的选择则应该根据肘部法则或者根据实际情况进行选择。

算法代码步骤实现

$\qquad$输入：样本集$D = { x_1,x_2,\dots,x_m }$

$\qquad$代码流程

  1. 初始化样本类中心点，从样本集$D$中选取簇数K个样本中心点$\mu_k$

  2. 计算样本点到各个类中心点$\mu_k$的距离$d^{(i)} := ||x^{(i)} - \mu^{k}||_2$

  3. 选择样本点到各个类中心点的最小距离并把样本点划入相应的簇数$c^{(i)} := j \quad minimizes \quad d^{(i)}$

  4. 重新计算类中心点$\mu_k:=\frac{1}{|C_k|} \sum_{i}^{C_k} x^{(i)}$

  5. 更新类中心点然后返回步骤2进行循环（最好设置迭代次数iters，防止进入死循环）

$\qquad$输出：各个簇类数据集$c^{(i)}$

Densit-Based Spatial Clustering of Applications with Noise基于密度聚类算法分析

算法原理分析

$\qquad$基于密度的聚类算法假设聚类结构能通过样本分布的紧密程度确定。密度聚类算法从样本密度的角度来考察样本之间的可连接性，并基于可连接样本不断扩展聚类簇数来获取最终的聚类结果。

$\qquad$算法主要基于一组领域(neighborhood)参数($\epsilon$,$MinPts$)来刻画样本分布的紧密程度，给定样本集$D = { x_1,x_2,\dots,x_m }$，领域($\epsilon$,$MinPts$)如图$(1)$所示，虚线圆圈表示其核心样本点的领域。

  • $\epsilon$-领域：领域包含样本集D中与$x_j$的距离不大于$\epsilon$的样本点，记$N_{\epsilon}(x_j) = { x_j \in D | dist(x_i,x_j) \leq \epsilon }$

  • $MinPts$-最小样本点数：用于判断$x_j$是否为样本核心对象还是噪声点，对于$|N_{\epsilon}(x_j)| \geq MinPts$的样本点，划入样本核心对象集，对于$|N_{\epsilon}(x_j)| \leq MinNum$的样本点，划入噪声点，一般$MinNum \leq MinPst$

算法代码步骤实现

$\qquad$DBSCAN算法先从数据集$D$中任选一个样本点作为核心对象（种子），从种子开始出发确定相应的聚类簇，根据给定的领域参数($\epsilon$,$MinPts$)找出所有的核心对象，由此重复迭代，遍历完全部的样本点为止。

$\qquad$输入：样本集$D = { x_1,x_2,\dots,x_m }$
$\qquad \ \ \qquad$领域参数($\epsilon$,$MinPts$)

$\qquad$代码流程

  1. 初始化簇类$C_1$

  2. 遍历数据集$D$，寻找核心对象$x_j$并确定其$\epsilon$-领域$N_{\epsilon}(x_j)$

  3. 判断$|N_{\epsilon}(x_j)| \geq MinPts$，将样本$x_j$加入到簇类集合$\Omega = \Omega \cup { x_j }$，并对其进行领域(neiborhood)扩展，否则不进行领域扩展；判断$|N_{\epsilon}(x_j)| \leq MinNum$，将$x_j$划入噪声点集合$N$

  4. 遍历$x_j$领域中的样本点，将样本点划入簇类$C_1$重复步骤2，进行同一簇类的领域扩展，领域扩展完成后，新增簇类$C_2$

  5. 将遍历过或划入噪声的样本点进行标记，对未标记的样本点$x_j \in D$重复步骤1，直至样本集$D$遍历完成

$\qquad$输出：各个簇类数据集$C = { C_1,C_2,\dots,C_i }$
